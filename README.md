# Employee Management System in C++

This is a simple C++ Object-Oriented Programming (OOP) project that demonstrates how to manage employee data using classes and constructors.

## Overview

The program creates and manages employee records using an `Employee` class. It demonstrates different ways of creating objects using constructor overloading.

## Features

- Store employee information (Name, ID, Department, Position)
- Default constructor
- Constructor overloading (multiple constructors)
- Getter and Setter functions
- Encapsulation (data hiding)
- Display employee details

## OOP Concepts Used

- Class and Object
- Encapsulation
- Constructors (Overloading)
- Getter and Setter methods
- Data abstraction
